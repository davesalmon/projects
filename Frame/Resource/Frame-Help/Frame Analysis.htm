Plane-Frame Analysis

About Plane-Frame Analysis

Overview

Plane linear frame analysis consists of discretizing a plane structure into beam and column segments (elements), connected to each other at joint points (nodes). Each node can move in 3 directions (degrees of freedom): horizontally (x), vertically (y), and rotationally (θ). Plane-Frame Analysis assumes that the behavior of the structure remains elastic throughout the deformation, i.e., if the loads are removed the structure will return to its original shape.

After defining the nodes and elements for the frame, structural properties (modulus of elasticity, area, and moment of inertia) must be assigned to each element. Additionally, a sufficient number of degrees of freedom must be restrained to prevent the structure or portions of it from moving as a rigid body.

After the structure has been defined, load cases are assigned consisting of node and element loads to be applied to the structure.

Based on the structure definition and the defined load cases, the structure can be analyzed and the internal forces (shear, moment, and axial force) and the displacements can be determined. Plane-Frame Analysis shows these results graphically superimposed on the structure.

Analysis method

Each unconstrained degree of freedom can potentially move due to the forces applied to the structure. This motion is resisted by the elements proportionally to their respective stiffness. Plane-Frame Analysis assumes that the behavior of each of the members and their connections to each other remain elastic throughout the deformation.

A set of linear simultaneous equations is constructed, one for each unrestrained degree of freedom.

[K]{x} = {b}

where [K] is the stiffness matrix, {x} is a vector of deformations in each unconstrained degree of freedom, and {b} is the load vector.

Each element in the structure contributes some resistance to deformation in each of the degrees of freedom to which it is attached. The combination of these resistances forms the stiffness matrix.

The loads applied to the structure contribute to the load vector. Loads may be applied directly to the nodes, or may be distributed along the length of the elements. In addition, pre-stress within elements due to thermal loading or pre tensioning may also contribute to the load vector.

After the equations have been constructed, they are solved for the deformations in each degree of freedom. These deformations are subsequently used to determine the internal forces (bending moment, shear force, and axial force) present in each element.

Creating a structure

Tools panel

A structure is created using the tools from the Tools panel (Fig 1). The node creation tool is used to create new nodes for the structure. When this tool is active, any click in the main view will create a new node point at that location. A grid is optionally provided (see Workspace) to assist in locating these nodes.

The created nodes are connected into a structure using the element creation tool . The element creation tool will create a new element between a selected node and either an existing node if one is clicked on, or a new node if the clicked location is not near any existing node. Holding the option key will temporarily change the element creation tool into the selection tool to allow quickly changing the selected node while creating the structure.

Figure 1. Tools panel.

The restraint assignment buttons in the second box within the tools window are used to restrain various degrees of freedom. Proceeding clockwise from the top-left they are: no restraint; restrain θ only; restrain x and θ; restrain y and θ; restrain x, y, and θ; restrain x and y, restrain, y, and restrain x.

Structure units

The units menu in the tools window can be used to select the units to use for the structure. There are three choices: Consistent, English, and SI.

Consistent units imply that the units used for force, distance, area, moment of inertia, and modulus of elasticity are all consistent. For example, pound, inch, inch2, inch4, and pounds/inch2. Such units are typically used when only the relative stiffnesses between the various structural elements are known.

English units are the standard set of units used in the United States: Forces in kips, dimensions in feet, moment of inertia in inch4, modulus of elasticity in ksi, area in inch2, and displacement in inches.

SI (international) units are the standard units in most of the rest of the world. Forces in kN, dimensions in m, moment of inertia in mm4, modulus of elasticity in kPa, and displacements in mm.

Note that the units for any specified quantity entered may be specified along with the value for that entity. The application will convert the value into the required units.

Workspace

The workspace is the coordinate system scaling and grid spacing used when defining a structure. It can be modified using the workspace panel (Fig. 3).

Figure 3. Workspace panel.

The world dimensions define the coordinate system of the window in which the structure is created. The world dimensions should be selected such that the structure being analyzed fits within the bounds. By default a window with 40 units in each direction is created with gridlines spaced one unit apart.

An optional grid may be drawn behind the structure. The spacing of the grid is set in the workspace panel. The checkboxes are used to specify if the grid is visible and if grid snap is enabled.

If grid snap is enabled, nodes added or selected within a radius of 25% of the grid spacing to a grid intersection will be assumed to have ocurred on the grid. This allows accurate placement and selection of nodes.

The scale defines the mapping between the screen and the world. For the workspace shown in Fig. 3, the world is 20 meters in width and height, and one inch on the screen corresponds to 1.0 meter. The scale must be sufficiently large that the world dimensions completely fill the window.

Selecting nodes and elements

The selection tool  is used to select nodes or elements.

Clicking on an individual node or element will select it. Nodes or elements may be added or removed from the current selection by holding the shift-key while selecting.

Multiple nodes (and their connecting elements) can be selected by clicking and dragging to select everything within the drag rectangle. Holding the shift key while dragging will add or remove nodes from the current selection.

Elements (without nodes) may be selected by holding the option key down while dragging across a portion of the structure. Any elements intersecting the drag rectangle will be selected (or added/removed from the selection if the shift key is down).

Copy and Paste

One quick method of creating a larger structure is to construct a representative sample of it and to use copy and paste to generate the rest. When node and elements are pasted into the structure they originated from, they retain all the properties and loads from the original copy. This means you can create a representative bay of one floor of the structure and copy that to generate all the other bays.

When a portion of a structure is placed such that one or more nodes overlap existing nodes, a dialog is presented asking if the nodes should be combined. Combining the nodes removes all the overlapping nodes and elements (Figure 4).

Assigning element properties

Element properties are assigned using the element properties window (Figure 5).

Figure 5. Element property definition.

Each row in the element property table defines a set of element properties that can be attached to one or more elements in the structure. The properties for each element are:

Modulus (of Elasticity): The inherent stiffness of the material being used.

Moment of Inertia: The second moment of area of the cross-section of the element, providing resistance to bending.

Area: The area of the cross-section of the element providing resistance to stretching.

Start Pinned: Checked if the start of the element is attached to its node with a pin connection.

End Pinned: Checked if the end of the element is attached to its node with a pin connection.

To attach a property definition, select the set of elements for each property type, then select the desired property row and press the attach button.

When a property row is selected and the Properties panel is in front, the elements attached to the selected property set are highlighted.

Figure 6. Highlighted property set.

Assigning node and element loads

Loads may be applied to the structure either directly to nodes or distributed along the length of the elements. Typically loads are divided into several categories including dead load, and one or more live load case.

Dead loads are loads that are typically well known such as the weight of the structure itself or the weight of permanent fixtures in the structure.

Live loads consist of people, things, wind, seismic loads, etc that may be applied to the structure during its life. These loads are typically not well known.

Loads are described using the Node Loads (Fig. 7) and Element Loads (Fig. 8) lists. You can create as many different load configurations as you want and assign each configuration to one or more elements.

Figure 7. Node load panel.

Each button in the node load list can be clicked to toggle between point forces applied to a node, and specified displacement. It makes sense only to applied specified displacement to degrees of freedom that are constrained. Such specified displacements can be used to evaluate the structure subject to support settlement.

Figure 8. Element load panel.

The rows in the element load list define load cases to be applied to the elements in the structure. For example, the weight of a beam or the pressure of the wind on the side of the frame would be defined as a lateral distributed load. Thermal loads can be applied as pre-force, while pre-moment could be used to simulated post-tensioning of concrete beams.

Figure 9. Attached loads.

Node and Element Inspectors

Node and Element inspector panel are provided to inspect the details for each node point or element. Typical node and element inspector panels for the sample structure are shown in Figure 13.

Figure 13. Inspector Panels.

Selecting one or more elements or nodes in either inspector selects the corresponding elements or nodes in both the main structure window and in the results table (for elements). The inspector shows a comprehensive summary of the characteristics of each node or element for the selected load case.

In the case of nodes, the inspector shows the location and applied point forces for the selected load case. If an analysis has been run, the node inspector also shows the reaction forces for restrained degrees of freedom and displacement for unrestrained degrees of freedom.

In the case of elements, the inspector shows the length and applied distributed forces for the selected load case. If an analysis has been run, the element inspector also shows the internal forces (axial, shear, and moment) at each end of the element, the maximum internal forces for the element, and the lateral displacement at each end and at the center of the element.
